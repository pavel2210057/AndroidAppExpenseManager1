Steps For transfer backup to any mobile or for transfer backup to any phone:

Step: 1

Take latest backup from "Local Backup" section in old phone.

Step: 2

Copy that backup folder (Internal storage root folder) from old phone and transfer it to new phone (Internal storage root folder).

Step: 3

After transferring backup folder to new phone now you can see backup list into "Local Backup" section.

Step: 4

Restore your latest backup.

Note: You can also put your local backup into safe place like Your Drive, Cloud Storage, Hard drive etc. Whenever you need to restore just put backup into internal storage root folder.



** Please don't change the folder name (/ExpenseBackup).
